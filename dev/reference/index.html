label_boundary_faces!(mesh::PMesh;physical_name="boundary")

Update mesh inplace by using partition ownership of faces to label only the boundary of meshes in a parallel mesh where the boundary is defined as a face owned by a given partition and not incident with any other faces on another partition (i.e., not on the interface).

GalerkinToolkit.options — Method
options(;kwargs...) -> Options

Create an object representing the default options for the current simulation. This object can be used as an optional argument in several object constructors in GalerkinToolkit, such as the mesh constructors cartesian_mesh and mesh_from_gmsh. In this case, the computations using the generated mesh, will use the given options by default.
